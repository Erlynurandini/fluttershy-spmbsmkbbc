# SMK Budi Bakti Ciwidey — Website PPDB

Website landing page interaktif untuk mendukung kegiatan PPDB SMK Budi Bakti Ciwidey Tahun Ajaran 2026/2027.

---

# 📌 Project Overview

Project ini dibuat sebagai media promosi digital sekolah dengan konsep modern, responsive, dan interaktif. Website menampilkan informasi sekolah, visi misi, keunggulan, serta jurusan yang tersedia di SMK Budi Bakti Ciwidey.

Website menggunakan sistem horizontal slider agar pengalaman pengguna terasa lebih modern dan menarik.

---

# 🎯 Goals

- Memperkenalkan SMK Budi Bakti Ciwidey
- Menampilkan informasi PPDB secara menarik
- Menampilkan jurusan BRP, PPLG, dan DKV
- Meningkatkan daya tarik calon peserta didik baru
- Mempermudah akses menuju formulir pendaftaran online

---
